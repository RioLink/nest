# Auto News Lab

Static automotive magazine project built to match the uploaded specification.

## Included

- semantic HTML5
- CSS3 + Vanilla JS
- cookie-gated ad loading
- 20+ article pages
- SEO meta tags, canonical tags, Open Graph, JSON-LD
- robots.txt and sitemap.xml
- article template for quick scaling

## Production notes

1. Replace `https://https://example.com` with the live domain in canonical URLs and sitemap.
2. Replace the AdSense client ID in `js/ads.js`.
3. Replace `mailto:admin@https://example.com` with your real contact flow.
